## 3. Reusable section modules

- **Hero block** — H1, subhead, primary + optional secondary CTA.
- **Problem block** — “Why X hurts” or “Signs that…” list; bridge to solution.
- **Solution block** — 3-pillar or benefit grid; cards with icon, title, short copy.
- **Process block** — 3–4 numbered steps; title + one-line description.
- **Path selector** — Launch / Rebrand / Marketing Assets → respective pages.
- **FAQ block** — Accordion; 3–5 Q&A; concise answers.
- **CTA block** — Headline, one line, primary button, optional secondary link.

## 5. Tone & hierarchy

- **Tone:** Premium, clear, strategic, authentic; avoid filler and generic agency speak.
- **Hierarchy:** One clear hero message per page; section order: Problem → Solution → Process → Proof → CTA.
- **Spacing:** Consistent vertical rhythm; alternate section backgrounds; max width for text (e.g. max-w-3xl).
